# 项目元信息 (Meta)

## 基本信息

**项目名称**: 如故（as old）

**项目标语**: 帮助内向者找到找到一见如故的朋友。

**版本**: 0.1.0

**创建日期**: 2025-12-24

**负责人**: 江浩

## 项目描述

内向者有连接他人的需求，但是难以主动联系。应用存储用户资料，通过向量数据库和AI找到和用户的需求、供给、兴趣爱好等匹配的人，并且提供破冰话题。

## 目标用户

- 用户群体: 想认识新朋友，但是难以主动连接他人的人。

## 核心价值主张

1. 存储用户资料。
2. 根据资料匹配用户和其他用户。比如，兴趣爱好相似、需求和提供匹配。
3. 为用户推荐匹配的其他用户。
4. 为用户提供破冰话题。

## 技术栈

**前端**:
- 框架: Next.js 15
- UI: Tailwind CSS + shadcn/ui
- 语言: TypeScript

**后端**:
- 运行时: Bun / Node.js
- 数据库: Milvus
- Milvus SDK：Node.js

**基础设施**:
- 托管: Vercel / EdgeOne Pages
- 数据库: Milvus / Zilliz cloud
- 认证: Better Auth

**AI集成**:
- AI SDK: Vercel AI SDK
- 模型接口: openai
